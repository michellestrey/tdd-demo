# TDD Demo

Projeto desenvolvido para prática de Test Driven Development (TDD), com foco em melhorar a lógica de programação, estrutura de código e qualidade através de testes automatizados.

## 📌 Objetivo

- Praticar desenvolvimento orientado a testes (TDD)
- Melhorar raciocínio lógico
- Treinar ciclos: Red → Green → Refactor
- Explorar boas práticas de organização de código

## 🧪 Tecnologias

- Java
- JUnit (para testes)
- IDE (Eclipse / VSCode)

## 🔁 Ciclo TDD

1. Escrever um teste que falha (Red)
2. Implementar o código mínimo para passar (Green)
3. Refatorar o código mantendo os testes passando (Refactor)

Obs: Baseado em estudos a partir da leitura do livro TDD do autor Kent Beck.
